# The Terrific WordPress Theme - A modular approach

## Included
* TerrificJS 1.0
* Modernizr 1.0.6
* JQuery 1.6.2

## JQuery Plugins (optional)
* Appear 1.1.1 (for async share buttons)
* Fancybox 1.3.4 (lightbox)
* Validation 1.8.1 (for comment form validation)
